# UI Prototypes

Ten alternative UI designs for the Open Volunteering feed page. Each prototype is built with vanilla HTML, CSS, and JavaScript.

## Prototypes

| Folder | Style | Description |
|--------|-------|-------------|
| `prototype_1_minimal` | Minimal | Clean layout, Libre Baskerville + DM Sans, warm neutrals, generous whitespace |
| `prototype_2_cards` | Cards | Card-based layout with avatars, tags, and hover effects (Inter) |
| `prototype_3_sidebar` | Sidebar | Dark sidebar navigation, dashboard-style layout (Source Sans 3) |
| `prototype_4_compact` | Compact | Dense, list-style layout with sidebar, efficient scanning (IBM Plex Sans) |
| `prototype_5_bold` | Bold | Dark theme, orange accent, strong typography (Space Grotesk + Work Sans) |
| `prototype_6_newsletter` | Newsletter | Magazine-style layout with section titles, Lora + Open Sans, green accent |
| `prototype_7_mobile` | Mobile | Mobile-first, phone frame, bottom navigation, FAB (Outfit) |
| `prototype_8_timeline` | Timeline | Vertical timeline with connecting line and node markers (Plus Jakarta Sans) |
| `prototype_9_split` | Split | Split view: feed on left, upcoming events & positions on right (Figtree) |
| `prototype_10_gradient` | Gradient | Soft gradients, glassmorphism, rounded corners, pink/purple accent (Nunito) |

## How to view

Open each `index.html` in a browser or serve via a local server (e.g. XAMPP).

```bash
# Example: open in browser
start ui_prototypes/prototype_1_minimal/index.html
```

## Features

Each prototype includes:
- Feed page with sample posts
- Navigation (Feed, Calendar, Map, Profile)
- Create post area
- Like / Comment / Share actions
- Pagination link
